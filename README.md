💬 Gemini Chatbot (WhatsApp-Style UI)
A modern web-based chatbot powered by Gemini 2.5 Flash API, built using Node.js and JavaScript, with a sleek WhatsApp-inspired UI using HTML & CSS.
This project demonstrates how to integrate a powerful AI model into a lightweight, responsive chat interface for real-time interaction.

------------------------------------------------------------------------------------------
🚀 Features

⚡ Fast AI responses using Gemini 2.5 Flash

💬 Clean WhatsApp-style chat interface

🌐 Full-stack JavaScript (Node.js + Frontend)

🎨 Responsive and modern UI design

🔄 Real-time message handling

🧠 Context-aware AI conversation

-------------------------------------------------------------------------------------------
🛠️ Tech Stac Layer	Technology

Backend	Node.js (Express)

Frontend	HTML, CSS, JavaScript

AI API	Gemini 2.5 Flash

Styling	Custom CSS (WhatsApp UI Inspired)

-------------------------------------------------------------------------------------------
⚙️ Installation

Clone the repository

git clone https://github.com/bayuaditya7/gemini-chatbot-api.git
cd gemini-chatbot

Install dependencies:

npm install

Create .env file:

GEMINI_API_KEY=your_api_key_here
PORT=3000

Run the server:

npm start

Open in browser:

http://localhost:3000
